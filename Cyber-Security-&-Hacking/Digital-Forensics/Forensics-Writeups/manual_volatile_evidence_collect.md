### CMD Windows

- Comandos de tiempo (Muy usado presedencia con otros comandos)

    - `date /?`
    - `date /t`
    - `time /?`
    - `time /t` 
    
- Network

    - `net /?` 
    - `net localgroup`  
    - `ipconfig /?`   
    - `ipconfig /all`
    - `route /?`
    - `route /?`
    - `route print`
    - `arp /?`
    - `arp -a`
    - `ipconfig /displaydns`
    - `netstat /?`
    - `netstat -a -b -n`

### SysInternals

- **TCP View - tcpvcon** (Network)

    - `tcpvcon /?`
    - `tcpvcon -a -n`
    - `tcpvcon -n`  

- **PS Tools - PsService.exe** (Services)

    - `PsService.exe /?`

- **driverview-x64** (Drivers)
